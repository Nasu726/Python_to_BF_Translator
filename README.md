## Python to BF Translator
AtCoderで動かせるBFコードの生成を目指しています。intは64bit固定です。

### 実装済み
- print(var: int | str)
- print(imm: str)
- assign(x: int | str)

### 直近で実装予定
- BinOp Arithmetic (ADD, SUB, MUL, FLOORDIV, MOD, POW)
- BinOp Logical (EQUAL, NOT_EQUAL, LT, LE, GT, GE)
- Boolean and Ops (True, False, AND, OR, NOT)
- input()
- int(input())
- if
- for i in range(n)

### 保留
- map(int, input().split())
- input().split()
- List
- Set
- Dict
- TrueDiv
- function
- XOR
